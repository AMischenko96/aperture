# «Aperture» 

## О проекте
Сайт для портфолио. 

## Технологический стек и функциональность:
### HTML
* Семантические теги

### CSS
* Flexbox
* Grid-layout
* Позиционирование блоков и элементов
* Адаптивная верстка под размеры экранов 1280px, 1024px, 768px, 320px
* БЭМ Nested
* Псевдоклассы CSS
* Адаптивное меню
